# Professional Portfolio Website

A clean, modern personal professional website to showcase your experience, skills, and projects.

## Features

- Responsive design that works on all devices
- Smooth scrolling navigation
- Animated sections on scroll
- Professional layout with hero section
- Experience timeline
- Skills showcase
- Projects portfolio
- Contact section with social links

## Customization

1. **Personal Information**: Update `index.html` with your:
   - Name and title
   - About section content
   - Work experience
   - Skills and technologies
   - Projects
   - Contact links (email, LinkedIn, GitHub)

2. **Colors**: Modify CSS variables in `styles.css`:
   ```css
   :root {
       --primary-color: #2563eb;
       --secondary-color: #1e40af;
   }
   ```

3. **Sections**: Add or remove sections as needed in `index.html`

## How to Use

Simply open `index.html` in your web browser to view the site locally.

## Deployment

You can deploy this site to:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting service

Just upload all files to your hosting provider.
